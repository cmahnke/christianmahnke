Kolophon

Diese Seite wurde mit Hugo generiert.

Zur Einbindung von Videos kommt das Theme hugo-video zum Einsatz. Als Schriftarten werden Josefin Sans, League Spartan und Inconsolata eingesetzt.

Folgende Javascript und CSS Frameworks und Bibliotheken kommen zum Einsatz:

  • chroma.js
  • jQuery
  • js-cookie
  • Normalize.css
  • Enter-view.js
  • OpenLayers
  • wordcloud2.js

Einträge mit 3D Inhalten nutzen:

  • A-FRAME
  • Three.js
