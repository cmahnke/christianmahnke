Maps for my Projektemacher blogs

Sunday, January 1, 2023

Another addition from Projektemacher Labs:

Preparations

Before posts can be displayed on maps, the following steps must be taken:

  • Geocoding of posts and / or tags
  • Generation of an application-specific representations (GeoJSON, KML)

After this has been done, the next step can be approached.

An example of the old 3D images can be found here.

Plotting data on a map in a browser

The map material provided by OpenStreetMap can be used to display the generated data in the browser. An external server generates the images needed to display a map.

External Content

External content is included at this point. Personal data might be transmitted to third-party platforms.
