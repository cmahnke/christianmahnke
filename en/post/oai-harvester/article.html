I didn’t want to write the millionth [OAI-PMH harvester]( OAI-PMH Harvester1)….

…so I took an existing2 and packaged3 it as a Docker image. The package also includes Saxon4 to be able to process XML metadata files…

The whole thing can be used to automatically add data to a Docker image.
