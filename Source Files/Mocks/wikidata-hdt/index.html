<rdf-graph>

Visualisiert RDF-Daten aus einer HDT-Datei als interaktiven Graphen.

<rdf-graph
      src="daten.hdt"
      sparql-query="SELECT ?s ?p ?o WHERE { ?s ?p ?o } LIMIT 100"
      languages="de,en"
      node-scaling="log"
      wikidata="labels"
      auto-execute
      export-svg
    ></rdf-graph>

Attribute

Attribut Typ Standard Beschreibung
src String — URL der HDT-Datei die geladen werden soll.
sparql-query String — SPARQL-SELECT-Query die auf dem geladenen Store ausgeführt wird. Muss die Variablen ?s, ?p, ?o binden.
languages String de,en Kommagetrennte Sprachpräferenz für Labels, z.B. de,en,fr. Wird für Store-Abfragen, Wikidata-Labels und Wikipedia-Links verwendet. en wird immer als letzter Fallback angehängt.
node-scaling off | linear | log off Skaliert Knotengröße anhand der eingehenden Verbindungen (Indegree).
  • off – alle Knoten gleich groß
  • linear – Größe proportional zum Indegree
  • log – Größe logarithmisch skaliert (empfohlen bei großen Graphen)
wikidata off | labels | incoming off Steuert das Verhalten bei Wikidata-Knoten.
  • off – keine Wikidata-Anreicherung
  • labels – Labels und Beschreibungen werden beim Aufbau des Graphen von der Wikidata-API nachgeladen
  • incoming – Klick auf einen Wikidata-Knoten lädt alle Entitäten die auf diesen Knoten verlinken und fügt sie dem Graphen hinzu
auto-execute Boolean false Führt die SPARQL-Query automatisch aus sobald die HDT-Datei geladen ist, ohne dass eine weitere Eigenschaftsänderung nötig ist.
export-svg Boolean false Blendet einen SVG-Export-Button ein. Der Button ist deaktiviert solange der Graph noch nicht bereit ist. Sichtbarkeit kann alternativ über :host(:not([export-svg])) #export-btn { display: none } im eigenen CSS gesteuert werden.

Events

Event Detail Beschreibung
store-loaded { quadCount: number } HDT-Datei wurde erfolgreich geladen und in den Store überführt.
load-error { error: string } Fehler beim Laden der HDT-Datei.
graph-ready { cy, quadCount } Graph wurde zum ersten Mal aufgebaut.
graph-updated { cy, quadCount } Graph wurde nach einer Einstellungsänderung neu aufgebaut.
query-error { error: string } Fehler beim Ausführen der SPARQL-Query.
node-selected Cytoscape Node-Daten Ein Knoten wurde angeklickt.
node-deselected — Klick auf den Hintergrund, Auswahl aufgehoben.
node-incoming-loaded { nodeId, qid, count } Eingehende Wikidata-Verlinkungen wurden geladen und dem Graph hinzugefügt.
svg-exported { filename: string } SVG wurde exportiert und der Download gestartet.

JavaScript API

Methode / Getter Rückgabe Beschreibung
exportAsSvg() void Exportiert den aktuellen Graphen als SVG-Datei und startet den Download.
getCy() cytoscape.Core | null Gibt die interne Cytoscape-Instanz zurück für direkten Zugriff.
getStore() WikidataStore | null Gibt den internen WikidataStore zurück für SPARQL-Queries und Wikidata-Anreicherung.
quadCount number Anzahl der Triples im geladenen Store.

Knotentypen

CSS-Klasse Farbe Beschreibung
post Rot, abgerundetes Rechteck Nicht-getaggte Subjekt-Ressourcen, typischerweise Blog-Posts oder Dokumente.
tagged Gelb, Raute Ressourcen mit einem projektemacher-Tag (schema:PropertyValue mit propertyID = "projektemacher" und value = "tag").
wikidata Blau, Ellipse Knoten deren URI auf wikidata.org/entity/Q… zeigt.
blank Grau, Raute RDF Blank Nodes.
uri Grün, Kreis Alle anderen benannten Ressourcen.
