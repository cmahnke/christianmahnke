GeoIP Tool

Sonntag, 30. April 2023

Kibana Visualisierung mit Raumplänen braucht Daten, z.B. von Geräten in einem Gebäude. Aber irgendwo müssen die Daten herkommen…

Eine Möglichkeit das zu tun ist einfach den GeoIP Prozessor von Elastic Search dafür zu benutzen. Leider ist es mit mitgelieferten Bordmittel nicht möglich eine eigene entsprechende Datenbank anzulegen. Daher habe ich dafür ein kleines Werkzeug geschrieben.

Mit dem Tool kann man eine Datenbank anlegen, die die Koordinaten von Geräten innerhalb eines Gebäudes schon bei der Indexierung z.B. einem Log Eintrag zuweist. Beispiele zur Nutzung finden sich bei GitHub.
