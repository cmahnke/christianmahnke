OAI-PMH Harvester

Sonntag, 16. April 2023
Christian Mahnke0009-0003-1361-4159

Schlagworte:
  • Daten

Den millionsten OAI-PMH Harvester1 wollte ich nicht schreiben,…

…daher habe ich einen bestehenden2 genommen und als Docker Image verpackt3. Im Paket ist auch Saxon4 enthalten um XML Metadatendateien weiterverarbeiten zu können…

Das Ganze kann verwendet werden um Daten automatisiert einem Docker Image hinzuzufügen.

Verwandte Beiträge:
  • Daten Container
  • Normdaten für private Sammlungen (und kleine Kultureinrichtungen)
  • JSON für Anwendungsbündel komprimieren
  • Vektor Kartenmaterial mit GitHub bauen
  • GeoIP Tool
